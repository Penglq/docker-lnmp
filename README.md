### 自建lnmp+redis



| 目录位置       | 功能        | 备注                               |
| ---------- | --------- | -------------------------------- |
| /web/data  | 存放项目      |                                  |
| /web/logs  | 存放日志      |                                  |
| /web/conf  | 存放配置文件    | nginx路径暂定为/web/conf/nginx/conf.d |
| /web/mysql | mysql数据目录 |                                  |
| /web/redis | redis数据目录 |                                  |
